## Customer Churn Prediction
**Question:** Which telecom customers are likely to leave?

**Approach:** EDA → Feature Engineering → 
Logistic Regression + Random Forest

**Key findings:**
- Best model AUC: 0.836
- Top churn drivers: Total charges, tenure, 
  monthly charges, payment method, fiber optic
- Month-to-month customers churn at 3x the rate 
  of annual contract customers

**Tools:** Python, pandas, scikit-learn, 
matplotlib, seaborn

**Dataset:** IBM Telco Customer Churn (Kaggle)
